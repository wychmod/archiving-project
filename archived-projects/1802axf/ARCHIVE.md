## 项目简介

爱先蜂(axf 取自"爱先蜂"拼音首字母)是 1802 期 Django 课程的课设项目,模仿"爱鲜蜂"模式,实现一个 O2O 闪送超市电商 Demo。项目分四个主要页面:主页(轮播图/导航/必买/便利店)、闪送超市(商品分类与筛选)、购物车(增删改查与下单)、我的(登录/注册/收货地址/订单/评价/文件上传)。

## 技术栈

- Python
- Django 1.11.4
- SQLite(Django 默认数据库)
- Django 模板系统 + 模板继承
- jQuery 3.1.1
- Bootstrap 3.x
- Swiper(轮播图)
- HTML5 / CSS3

## 学习重点

- Django 项目结构与 MTV 模式:Models、Views(基于函数的视图)、Templates
- 自定义用户模型:继承 `AbstractUser` 扩展 `icon` 字段
- 抽象基类 `Common` 抽离轮播图、导航、必买、便利店等公共字段
- 一对多 / 多对多关系:商品类别与商品、用户与购物车、用户与订单
- 用户认证与会话管理:登录状态判断、登录拦截
- 模板继承与静态资源加载:`{% load static %}`、`{% block %}`、`{% url %}`
- 移动端 H5 适配:viewport、rem、底部固定 Tab Bar
